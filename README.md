# Meme Generator

it is a simple application that generates memes using event listener, `useState`, `useEffect` with Reactjs 

 ![Meme generator ](./src/images/Meme%20Generator.png "Meme generator")
## installation 
could use: 
- `npm install`
- `yarn`

## run the app
could use: 
- `npm start`
- `yarn start`
